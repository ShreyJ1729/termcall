# TermCall: FaceTime in the terminal

brew install termcall

launch on login in background to see if user active or not for status

if user calls then background process opens up a terminal window which shows who's calling

calls are both video/audio

figure out how video resolution/frame rate and audio bitrate modifications will work.

public/private key encryption for authentication

friend request feature - can see which friends are inactive/in call/active (red dot, yellow dot, green dot)

text chat feature to send messages to friends. on opening terminal if any messages are unread, termcall will notify x messages unread.
